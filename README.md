## Швидкий Старт

1. **Встановлення**
   ```
   Figma → Plugins → Browse plugins → Content Craft → Install
   ```

2. **Використання**
   ```
   Виберіть текст → Запустіть Content Craft → Виберіть тип контенту → Generate
   ```
